## About

MingMing helps you study with question banks, medical libraries, and flashcards in one place. Designed for focused, daily practice with a clean workflow.

- **Qbanks** — Practice questions with organized categories
- **Library** — Read article-based medical references
- **Flashcards** — Review decks for active recall
- **Offline-first** — Works fully without internet (except AI features)
- **AI-powered** — Built-in Gemini AI integration
